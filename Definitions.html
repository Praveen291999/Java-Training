DEFINITIONS

Definitions
JAVA
Java is an object-oriented programming language that produces software for multiple platforms.
Class
A class is a user defined blueprint or prototype from which objects are created.
A class has Data members and methods.
Object
An Object is an instance of a class.
Using object we can access data members of the class.

Literals
The values assigned to variable are calles as literals.

User Defined Datatypes
  1. Classes
  2. Interfaces
  3. Enum
Interface
An interface in Java is a blueprint of a class. It has static constants and abstract methods.
Using interface Multiple Inheritance can be acheived.

Enum
An enum is a special "class" that represents a group of constants (unchangeable variables, like final variables).

Primitive datatypes
Primitive datatypes are userdefined datatypes.
Example: byte, char, short, int, long, float, double and boolean.

User defined Datatypes
A user-defined array type is a data type that is defined as an array with elements of another data type.
Example:String,Arrays
CheckStyle
Checkstyle is a tool/Plugin that helps programmers for maintaining better coding standard.
